# 1-2 环境搭建 ： 双刀流

以下是对C++开发环境搭建指南的优化整理，特别强化了"双刀流"理念——即同时支持macOS和Windows平台的配置方案：

# 环境搭建核心思路

  1. 统一工具链：选择跨平台工具减少切换成本
  2. 版本控制：确保工具版本匹配（编译器/CMake/Qt）
  3. 配置即代码：所有环境设置脚本化保存
  4. 故障预防：路径校验+版本检查先行

# 搭建的核心思路

本教程旨在帮助读者搭建一个适用于 C++ 开发的环境，涵盖从代码编辑器到编译器的各个方面。以下是环境搭建的核心思路：

  1. 选择合适的代码编辑器：使用 Visual Studio Code 作为主要的代码编辑器，提供强大的扩展支持和调试功能。
  2. 配置 CMake：使用 CMake 作为跨平台的构建系统，简化项目管理和编译过程。
  3. 安装 Git：使用 Git 进行版本控制，方便代码管理和协作开发。
  4. 选择合适的 C++ 编译器：根据操作系统选择 GCC 或 Clang 编译器，确保代码能够正确编译和运行。
  5. 使用 C++ 标准库：利用 C++ 标准库提供的丰富功能，提高开发效率。
  6. 安装 Markdown 编辑器：使用 Markdown 编辑器编写文档和注释，便于项目文档化。
  7. 配置其他工具：根据需要安装 Docker、Postman、Figma 等工具，提升开发和协作效率。

# 搭建简要指南：

项目运行即可

安装 Xcode（xcode 组件包含c、c++和clang编译器）；安装Command Line Tools for Xcode （包含git）；拉取代码仓库 （安装gitlfs)；安装cmake ；获取对应的Qt源码库；配置cmake构建工程命令；开始运行；

简单的理解思路 ： 首先，就是要选择一个编辑器， 顾名思义，就是用来方便我们程序员修改代码的；

其次，下载对应的编译器， 这个根据OS来，核心就是把我们写的c++东东，能够正常编译和运行，就是能够控制OS。

再者， 最重要的就是把代码搞下来， 团队合作少不了git, 跨平台编译又需要cmake ;

最后，就是项目依赖的三方库，需要弄下来， 一般就是c++的官方库， Qt就是源码库。

还补充一点， 出问题三思路 ：1、 版本匹配了吗？（xxx --version） 2、软件路径找到了吗？(path 系统路径) 3、重来胜过排查问题，手法很重要

# 具体安装和步骤指南：

# 一.安装Xcode

  1. 官方下载地址：（搜索xcode下载即可）https://developer.apple.com/download/all/ (opens new window)
  2. os arm 13.0系统建议下载Xcode 14.1 以上 版本
  3. 下载匹配的Command Line Tools工具 （温馨提示：直接通过appstore下载的是最新版本，容易出现版本不匹配问题）

# 二.首次拉取代码

  1. 安装brew： /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh (opens new window))" 配置环境变量： （1）echo 'export PATH="/opt/homebrew/bin:$PATH"' >> ~/.bash_profile （2）source ~/.bash_profile
  2. 安装git lfs： Step 1 : brew install git-lfs Step 2: git lfs install Step 3:git lfs version
  3. 拉取代码仓库：git clone [Pssh地址]

# 三.安装cmake

  1. 直接官网下载3.21.0版本以上 （官网下载链接：https://cmake.org/download/#previous） (opens new window)
  2. 解压后如果无法使用，输入下方命令：
  3. sudo "/Applications/CMake.app/Contents/bin/cmake-gui" --install

（温馨提示：验证安装成功与否：cmake --version）

# 四.配置cmake工程 (Clion)

获取qt源码并配置对应路径：/Users/ws/Qt/5.15.2/clang_64 （温馨提示：采用ftp传递可以避免签名等问题） cmake -S . -B MacProject -G "Ninja Multi-Config" -DCMAKE_BUILD_TYPE=Debug -DCMAKE_OSX_ARCHITECTURES=arm64 -DPRJ_FOR_DEV=ON -DFILMORA_USE_ANTIPIRACY=OFF -DFILMORA_FOR_MIAOYING=ON -DFilmoraTest= -DcustomQtDir=/Users/ws/Qt/5.15.2/clang_64
